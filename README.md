# Diamond 💎✨

Welcome to **Diamond Jewellery Boutique** – Egypt's premier luxury gifting destination.

### 🔗 [Live Demo: View the Boutique Here](https://dareen-housam.github.io/Diamond/)

---

## 💍 Overview
Diamond is a high-end, responsive e-commerce landing page designed for a luxury jewelry brand. It features a sophisticated "boxed" layout, keeping the content elegantly centered with consistent framing, perfectly suited for a premium brand aesthetic.

## 🚀 Key Features
- **Modern Navigation:** A sleek, minimal header with mega-menu capabilities for "Shop" and "Gift Guide" categories.
- **Interactive Iconography:** Custom feature icons with a unique "sliding reveal" animation on hover.
- **Visual Overlap Design:** Category cards featuring 3D-layered imagery that creates depth and a premium magazine feel.
- **3D Best Sellers Grid:** A smooth, 3D-flip product gallery showcasing ratings, sale prices, and "View Details" interactions.
- **Luxury Hero Section:** A left-aligned hero call-to-action designed to complement high-quality brand photography.
- **Responsive Architecture:** Fully optimized for mobile, ensuring the 3D effects and image layouts scale beautifully on smaller screens.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Custom Flexbox and Grid layouts, 3D transforms, and complex hover animations.
- **Font Awesome:** Integrated professional iconography.

## 📁 Folder Structure
- `index.html` – The main boutique homepage.
- `css/` – Stylesheets including `style.css` and `all.css`.
- `photos/` – High-resolution jewelry assets and brand logos.
- `webfonts/` – Source files for typography and icons.
